# Personas

## Objetivo

Definir a base documental para futuras personas, necessidades, motivacoes e jornadas relacionadas ao DesenvolvePsi.

## Descricao

Este documento sera utilizado para organizar perfis de usuarios e stakeholders que orientarao arquitetura de informacao, conteudo, experiencia e priorizacao de funcionalidades. Nesta Sprint 00, apenas a estrutura documental foi criada.

## Indice das Futuras Secoes

1. Metodologia de definicao
2. Personas primarias
3. Personas secundarias
4. Necessidades principais
5. Barreiras e objecoes
6. Cenarios de uso
7. Criterios de priorizacao
8. Implicacoes para UX e conteudo
