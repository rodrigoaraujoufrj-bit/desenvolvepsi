# Master Prompt

## Objetivo

Centralizar diretrizes futuras para orientar prompts, agentes e execucoes assistidas no desenvolvimento do DesenvolvePsi.

## Descricao

Este documento servira como referencia para manter consistencia entre produto, UX, design, conteudo e implementacao quando houver uso de automacao ou assistentes de desenvolvimento. Nesta Sprint 00, ele define apenas a estrutura inicial.

## Indice das Futuras Secoes

1. Contexto do projeto
2. Objetivos de produto
3. Regras de tom e linguagem
4. Diretrizes de design
5. Diretrizes tecnicas
6. Restricoes de implementacao
7. Criterios de aceite
8. Historico de decisoes
