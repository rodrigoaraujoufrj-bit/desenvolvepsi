# FAQ - Layout

## Largura e grid

Usar container default ou largura de leitura entre 760px e 900px. A FAQ deve
ser centralizada para favorecer leitura sequencial.

## Alinhamentos

Titulo centralizado no desktop. Itens do accordion alinhados a esquerda para
leitura natural. No mobile, manter textos alinhados a esquerda.

## Espacamentos

Usar espacamento vertical generoso antes do accordion. Cada pergunta deve ter
area clicavel ampla, respeitando minimo de toque de 44px.

## Proporcoes

O accordion deve ocupar largura suficiente para respostas confortaveis, sem se
estender demais em telas grandes. Respostas longas devem quebrar em linhas
curtas e legiveis.

## Responsividade

- Desktop: bloco centralizado com largura controlada.
- Tablet: manter largura fluida com gutters.
- Mobile: accordion em largura total do container.

## Hierarquia visual

1. Titulo da secao.
2. Pergunta ativa.
3. Resposta aberta.
4. Perguntas fechadas.
5. Numeracao ou badge.

## Ordem dos elementos

As primeiras perguntas devem tratar as duvidas mais basicas sobre iniciar
terapia. Perguntas praticas e institucionais devem vir depois.
