# Footer

## Objetivo da secao

Encerrar a Landing Page com informacoes institucionais, navegacao auxiliar e
avisos de responsabilidade.

## Papel na jornada do usuario

O Footer funciona como base de confianca final. Ele deve oferecer caminhos
rapidos, contexto institucional e espaco para informacoes oficiais quando
estiverem definidas.

## Comportamento esperado

A secao deve ser discreta, clara e util. Nao deve competir com o CTA final nem
introduzir conteudo comercial novo.
